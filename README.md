 Название проекта
Furniture Store Management System
Система управления мебельным магазином — десктопное приложение для учета товаров, продаж и поставок.

 Краткое описание
Проект представляет собой полнофункциональную систему управления мебельным магазином с графическим интерфейсом. Приложение позволяет вести учет товаров на складе, управлять продажами, поставками и анализировать бизнес-показатели.

 Возможности системы
 Аутентификация и роли
Регистрация и вход в систему
Две роли пользователей:
Администратор — полный доступ ко всем функциям
Менеджер — работа с каталогом и продажами
 Управление товарами
 Каталог товаров — полный перечень мебели с фото и характеристиками
 Складской учет — отслеживание остатков и движения товаров
 Категории и фильтры — удобная навигация по ассортименту
 Бизнес-процессы
 Продажи — оформление заказов и чеков
 Поставки — прием товаров от поставщиков
 Поставщики — управление контрагентами
 Аналитика — отчеты по продажам и остаткам
 Требования к системе
Программное обеспечение
Python 3.10 или выше
PostgreSQL 12+
Git для клонирования репозитория
База данных
В PostgreSQL должны быть созданы следующие таблицы:
users — пользователи системы
products — товары (мебель)
categories — категории товаров
sales — информация о продажах
suppliers — поставщики
purchases — информация о закупках
stock — складские остатки
 Установка и настройка
1. Клонирование репозитория
bash
git clone https://github.com/ваш-репозиторий/furniture-store.git
cd furniture-store
2. Создание виртуального окружения
bash
python -m venv venv

# Активация для Windows (PowerShell):
venv\Scripts\activate

4. Настройка базы данных
Отредактируйте файл db.py:

python
DB_CONFIG = {
    "host": "localhost",
    "port": 5432,
    "database": "furniture_store",
    "user": "your_username",
    "password": "your_password",
}
5. Инициализация базы данных
bash
python init_db.py

# Запустите приложение
python main.py
Способ 2: Прямой запуск
bash
python main.py
 Сборка в исполняемый файл (EXE)
1. Установка PyInstaller
bash
pip install pyinstaller
2. Сборка приложения
bash
pyinstaller build_exe.spec
3. Готовые файлы
После сборки вы найдете:

 dist/furniture_store/ — папка с приложением

 dist/furniture_store.exe — исполняемый файл

 dist/furniture_store/photo/ — папка с фотографиями товаров
